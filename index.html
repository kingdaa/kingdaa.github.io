Largest Rectangle in Histogram

Given n non-negative integers representing the histogram’s bar height where the width of each bar is 1, find the area of largest rectangle in the histogram.


Above is a histogram where width of each bar is 1, given height = [2,1,5,6,2,3].


The largest rectangle is shown in the shaded area, which has area = 10 unit.

For example,
Given height = [2,1,5,6,2,3],
return 10.

Use a stack to process indices of histogram from left to right. If height of current histogram is more than height of top of the stack, push index of current histogram. If height of current histogram is less than or equal to height of top of the stack, pop the stack until stack is empty or height of top of stack is less than current histogram. On each pop, calculate the rectangle size that can be formed from popped histogram and see if it is max. (The size should be (current index of histogram - index of stack top histogram - 1) * height of stack top histogram). Push a 0 at last to the stack to make sure all the histograms are being processed.
